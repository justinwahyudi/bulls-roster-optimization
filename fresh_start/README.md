# Bulls Roster Fresh Start

Open this folder directly in VS Code, then start with:

- `PROJECT_CONTEXT.md` for the project framing, constraints, and roadmap.
- `phase2_fresh.ipynb` for the active analysis notebook.
- `free_agent_fit_2026.ipynb` for 2026 free-agent availability, salary context, and Bulls fit scoring.
- `draft_free_agency_scenarios_2026.ipynb` for testing how possible Bulls draft picks should reshape free-agent priorities.
- `prospect_measurements_2026.ipynb` for combining prospect stats, NBA combine measurements, and scouting/intangible grades.

This folder reuses the working virtual environment from the parent project:

```bash
../.venv/bin/python
```
